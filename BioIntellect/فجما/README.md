# 🏥 MedDash - Medical Dashboard System

A comprehensive medical dashboard system built with React, TypeScript, and Tailwind CSS featuring AI-powered analysis tools for healthcare professionals and patients.

## ✨ Features

### 👨‍⚕️ Doctor Dashboard
- **Statistics Overview**: Real-time patient statistics and performance metrics
- **Appointment Management**: View and manage daily appointments with timeline view
- **MRI Analysis**: AI-powered brain scan analysis with confidence scores and detailed findings
- **ECG Monitoring**: Real-time heart signal monitoring with AI analysis
- **Patient Records**: Complete patient information and medical history
- **Quick Actions**: Fast access to common tasks

### 👤 Patient Dashboard
- **Appointment Booking**: Easy appointment scheduling with calendar integration
- **Medical Records**: View test results and medical history (read-only)
- **Medication Tracker**: Track daily medications with reminders
- **Vital Signs**: Monitor health metrics and trends
- **Doctor Communication**: Message your healthcare provider
- **Health Tips**: Personalized health recommendations

## 🎨 Design Features
- Modern medical color scheme (Blue #2563EB, Green #10B981)
- Dark mode support
- Smooth animations with Motion (Framer Motion)
- Responsive design for all devices
- Glassmorphism effects
- Interactive charts and graphs

## 🚀 Technologies Used
- **React 18** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **React Router** - Navigation
- **Recharts** - Data visualization
- **Motion (Framer Motion)** - Animations
- **Radix UI** - Accessible components
- **Lucide React** - Icons
- **Sonner** - Toast notifications
